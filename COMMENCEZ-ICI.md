# 🚀 COMMENCEZ ICI - 3 étapes simples

## ✅ Votre portfolio est PRÊT !

### 📍 ÉTAPE 1 : Testez localement (2 minutes)

```bash
npm run dev
```

Ouvrez http://localhost:3000 dans votre navigateur.

### 📍 ÉTAPE 2 : Poussez sur GitHub (3 minutes)

1. **Créez un nouveau repository sur GitHub** :
   - Allez sur https://github.com/new
   - Nom : `portfolio`
   - Cliquez sur "Create repository"

2. **Poussez votre code** :
   ```bash
   git remote add origin https://github.com/tchiboub-dot/portfolio.git
   git branch -M main
   git push -u origin main
   ```

### 📍 ÉTAPE 3 : Déployez sur Vercel (2 minutes)

1. Allez sur https://vercel.com
2. Connectez-vous avec GitHub
3. Cliquez sur "Add New..." > "Project"
4. Importez `portfolio`
5. Cliquez sur "Deploy"
6. ✅ C'est fait ! Vous avez votre URL

### 🔗 Ajoutez l'URL sur LinkedIn

Copiez l'URL Vercel et ajoutez-la :
- Profil LinkedIn > "Contact info" > Website
- Section "About" > Ajoutez la ligne avec l'URL
